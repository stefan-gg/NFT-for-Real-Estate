# NFT-for-Real-Estate

This project is created for the Polkadot Prodigy hackathon


The only requirement for this project are secret keys for Filebase. You can obtain them by creating a Filebase account. However, there might be additional setup steps, so I recommend checking the link to the website I posted on my TaiKai project profile.


If you want to run the project locally you can clone the project and run these commands:
  cd nft-for-real-estate
  npm install
  npm start
